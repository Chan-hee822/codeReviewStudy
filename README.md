- 구현 기능 목록

  - 회원 ID로 조회 기능
    - 유효성 검증 : ID가 존재하지 않을 시 예외

  - 롬복 어노테이션 사용 않기
  - 계층형 아키텍쳐로 구현할 것

  - PostMan으로 성공 결과 캡쳐 해서 PR할 것
